# 日常话题 · 初阶答案说明

文档《日常话题逻辑构建》里**只有第 1 题**给出了明确答案：

- **展开逻辑排序**：C → B → D → A  
- **参考答案**：Yes, I like reading. It makes me feel relaxed and it's my favorite free-time activity. For example, I read a fiction last night.

所以**第 1 题的正确答案**是按文档来的：**C → B → D → A**（对应代码里的 `correctOrder: [2, 1, 3, 0]`）。

---

## 第 2–20 题：目前是「推断答案」

第 2 题到第 20 题，文档里**没有**写「展开逻辑排序」或参考答案，只有：

- 题目  
- A/B/C/D 四句  
- 连接词  

前端里的正确答案是我按**常见口语逻辑**推的：

- 先**观点/直接回答**（多为 A）  
- 再**原因/拓展**（B、C）  
- 最后**举例/个人经历**（D）  

所以目前第 2–20 题在代码里多数是 **A → B → C → D**（`correctOrder: [0, 1, 2, 3]`）。  
**这不是文档给的答案，需要你根据教学/教材再确认或修改。**

---

## 如何修改初阶答案

在项目里打开 **`logic-data.js`**，找到 `dailyBeginner`（或当前的 `dailyTopics`）数组。

每一题是一个对象，例如：

```js
{ question: '...', options: ['A句', 'B句', 'C句', 'D句'], connectors: '...', correctOrder: [0, 1, 2, 3] }
```

- **correctOrder** 表示正确顺序，用选项下标表示：  
  `[0,1,2,3]` = A→B→C→D，`[2,1,3,0]` = C→B→D→A。  
- 你确认某题的正确顺序后，把对应顺序改成 0–3 的排列即可。  
- 若某题文档后来补了「展开逻辑排序」或参考答案，按文档改 `correctOrder` 并保存。

改完后刷新前端，练习会按新答案判对错。
