# 如何运行项目

---

## 方式一：纯 HTML 版（无需 Node，推荐）

直接用浏览器打开 **`standalone.html`** 即可（双击或在浏览器里“打开文件”选这个文件）。

- **选择题、批改、口语倒计时、修改一次文本** 都能用。
- **语音转文字**：部分浏览器在「直接打开本地文件」时可能不允许用麦克风，若遇此情况见下方「用 Python 起一个简单服务器」。

### 若语音不工作（可选）

Mac 一般自带 Python，在终端执行（**不需要装 Node**）：

```bash
cd "/Users/simonezhang/cursor/interview练习"
python3 -m http.server 8080
```

然后用浏览器打开：**http://localhost:8080/standalone.html**  
按 `Ctrl+C` 可停止服务器。

---

## 方式二：React 版（需要 Node）

## 1. 在项目目录执行 npm install

**方法一：在 Cursor 里用终端**

1. 菜单栏点 **终端 (Terminal)** → **新建终端**
2. 在终端里输入下面两行（每行回车一次）：

```bash
cd "/Users/simonezhang/cursor/interview练习"
npm install
```

**方法二：在 Mac 自带的「终端」里**

1. 打开「终端」应用（Spotlight 搜 “终端”）
2. 输入：

```bash
cd "/Users/simonezhang/cursor/interview练习"
npm install
```

**说明**：`cd` 后面的路径就是你这个项目所在的文件夹路径；`npm install` 会安装依赖，需要联网。

---

## 2. 启动开发服务器

依赖装好后，在**同一个终端**、**同一目录**下执行：

```bash
npm run dev
```

终端里会出现类似：

```
  VITE v5.x.x  ready in xxx ms
  ➜  Local:   http://localhost:5173/
```

---

## 3. 在浏览器里打开

用浏览器打开上面显示的地址，例如：**http://localhost:5173**

不要直接双击打开项目里的 `index.html` 文件（会白屏），必须通过 `npm run dev` 启动后访问给出的地址。

---

## 如果还是白屏

1. 确认是用 **http://localhost:5173** 打开的，而不是 `file:///...`
2. 按 **F12**（或右键 → 检查）打开开发者工具，看 **Console** 里是否有红色报错
3. 若页面显示「页面出错了」和一段错误信息，把那段错误贴给开发者即可排查
