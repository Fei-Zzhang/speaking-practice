# 口语练习网站

本地运行：

```bash
npm install
npm run dev
```

在浏览器打开提示的地址（如 http://localhost:5173），建议使用 **Chrome 或 Edge** 以使用自带的语音识别（Web Speech API）。

---

## 功能说明

1. **多题目分区**：首页展示多个分区，每个分区可单独开始练习。
2. **选择题 → 批改 → 口语**：部分分区先做选择题，提交后显示批改结果，再进入「口语作答」。
3. **口语作答**：点击「开始答题」后 **45 秒倒计时**，使用浏览器 **语音转文字** 输出，可在时间结束后 **再修改一次文本**（共允许二次修改），然后提交。
4. **批改**：
   - **选择题**：在页面内直接对比正确答案，**不需要 API**。
   - **口语/文本**：若只做「提交并保存」，不需要 API；若要做 **智能批改（语法、内容、流利度等）**，需要接入 **批改 API**（见下）。

---

## 批改答案是否需要 API？

| 类型     | 是否需要 API | 说明 |
|----------|--------------|------|
| 选择题   | **不需要**   | 正确答案已在题目数据中，前端对比即可。 |
| 口语文本 | **可选**     | 仅保存/展示：不需要。要做评分、评语：需要接入大模型或第三方批改 API。 |

口语批改接口已预留：见 `src/api/grading.js`。在项目根目录新建 `.env`，配置：

```env
VITE_GRADING_API_BASE=https://你的批改服务地址
```

并实现后端 `POST /grade/speaking`，请求体 `{ sectionId, prompt, text }`，返回如 `{ score, feedback }` 即可。
